Detailed Description

Boost.Asio를 이용한 간단한 TCP 에코 서버 클래스.

지정된 포트 번호로 리슨(listen)하여 클라이언트의 연결을 받고, 받은 데이터를 그대로 다시 돌려주는 Echo 기능을 제공합니다.
